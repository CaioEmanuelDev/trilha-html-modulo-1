História dos computadores

(Voltar)

Os computadores surgiram a partir de máquinas de cálculo como o ábaco. Com o tempo, evoluíram para sistemas cada vez mais complexos. Um marco importante foi o ENIAC, considerado o primeiro computador eletrônico de uso geral.

"O ENIAC podia realizar 5.000 operações por segundo, um feito incrível para a época."

Saiba mais em Computer History Museum.

O que são Clients?

(Voltar)

Um Client é o dispositivo ou programa que solicita informações a um servidor. Exemplos: navegadores como Google Chrome ou Firefox.

HTTP é o protocolo usado para essa comunicação.

O que são Servers?

(Voltar)

O Servidor é responsável por responder às requisições dos clientes. Ele pode armazenar páginas, bancos de dados e arquivos.

Exemplo: um servidor web como Apache ou Nginx.

Veja mais em MDN Web Docs.

Como é uma estrutura básica de um HTML?

(Voltar)

Um documento HTML começa com a declaração <!DOCTYPE html>. Em seguida, temos as tags principais:

  1. <html>: raiz do documento
  2. <head>: informações como título e metadados
  3. <body>: conteúdo visível da página

Exemplo de código: 

    <!DOCTYPE html>
    <html>
      <head>
        <title>Minha Página</title>
      </head>
      <body>
        <h1>Olá Mundo!</h1>
      </body>
    </html>

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

Uma lista ordenada usa números:

  1. Primeiro item
  2. Segundo item

Já uma lista não ordenada usa marcadores:

  • Item A
  • Item B
